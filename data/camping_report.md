## エリア別候補地・市場調査

### 宇野港周辺
*コンセプト：直島フェリー客流・アート路線・海望キャンプ*

#### 市場動向・参考施設

**たまの湯キャンプ場**（既存運営施設・参考）

| 項目 | 詳細 |
|------|------|
| **所在地** | 岡山県玉野市、JR宇野駅徒歩3分 |
| **施設規模** | オートサイト（電源付き6,000円〜）+ グランピング（ロータステント20,000円〜/2名） |
| **特徴** | 瀬戸内海眺望・隣接温泉施設「たまの湯」利用可 |
| **海景・眺望** | 瀬戸内海望あり |
| **URL** | [施設詳細](https://www.seto-tamanoyu.jp/camp-top/) |

**SETONITE（セトニテ）**（既存高級グランピング・市場参考）

| 項目 | 詳細 |
|------|------|
| **所在地** | 岡山県玉野市、JR宇野駅よりタクシー約10分 |
| **施設規模** | ヴィラ型テント8棟、各棟約40㎡、シャワー・トイレ・エアコン付き |
| **開業** | 2022年12月 |
| **コンセプト** | 瀬戸内海望・バレルサウナ・BBQコース料理・アクティビティ |
| **海景・眺望** | 瀬戸内海目の前（絶景） |
| **市場単価参考** | 高単価グランピング（ヴィラ型） |
| **URL** | [施設詳細](https://setonite.jp/) |

> **投資示唆**：宇野港周辺はすでに高品質なグランピング施設が開業しており、直島フェリーアクセスを活かした観光需要が実証済み。追加用地の需要は高く、500坪以上の海望土地があれば開発ポテンシャル大。

### 王子が岳
*コンセプト：パノラマ海景・高単価グランピング・絶景サイト*

#### 市場動向

| 項目 | 詳細 |
|------|------|
| **エリア** | 岡山県玉野市・倉敷市境（王子が岳山頂周辺） |
| **特徴** | 瀬戸内海のパノラマ絶景、岩山・自然公園エリア |
| **グランピング参考** | 岡山県内グランピング施設は「グランピングヴィレッジ いぶきの里」等が先行 |
| **課題** | 自然公園区域の規制確認要、山林売地の流通少ない |
| **URL参考** | [岡山グランピング施設一覧](https://www.resort-glamping.com/accommodation_category/okayama/) |

> **投資示唆**：王子が岳は眺望の絶景性が際立つが、自然公園法・国定公園区域の規制可能性あり。土地取得前に都市計画・自然公園区域の確認が必須。山林・原野の大規模売地は地元仲介業者経由が現実的。

**参考問い合わせ先**
- 玉野市役所 建設部（開発許可・農地転用相談）
- 倉敷市 都市計画課
