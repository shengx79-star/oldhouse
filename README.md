# 日本瑕疵房产投资监控系统

## 概述

监控日本"再建築不可"等瑕疵房产的投资机会，支持东京23区全区域搜索。

## 搜索条件

| 项目 | 配置 |
|------|------|
| **区域** | 东京都23区（全区） |
| **价格上限** | 1500万円 |
| **类型** | 一戸建（戸建住宅） |
| **关键词** | 再建築不可、既存不適格、現状渡し、境界未確定、訳あり |
| **监控平台** | SUUMO、At Home、HOME'S、住友不動産ステップ |

## 数据文件

```
data/
├── properties.json      # 房源数据（JSON格式）
├── properties_report.md # 房源报告（Markdown格式，方便阅读）
config/
└── search_config.json   # 搜索配置
```

## 功能特性

- **自动搜索**: 每天早上9:00自动使用Tavily搜索房源
- **数据更新**: 自动更新 `properties.json` 中的房源列表
- **邮件报告**: 发现新增或价格变动的房源时通知
- **多平台覆盖**: SUUMO、HOME'S、住友不動産ステップ等主流平台

## 风险提示

| 关键词 | 风险说明 |
|--------|----------|
| 再建築不可 | 无法重建，贷款受限 |
| 既存不適格 | 不符合现行建筑标准 |
| 現状渡し | 按现状交付，可能存在未披露问题 |
| 境界未確定 | 土地边界不明确 |
